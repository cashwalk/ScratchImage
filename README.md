## Installation

#### CocoaPods (iOS 8+)

```ruby
platform :ios, '8.0'
use_frameworks!

target '<Your Target Name>' do
pod 'ScratchImage'
end
```

#### Carthage (iOS 8+)

```ruby
github "cashwalk/ScratchImage"
```
